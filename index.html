Nastavení Apps Script (pro zápis)

  1. Otevři Google Sheet → Rozšíření → Apps Script
  2. Smaž obsah, vlož kód níže, ulož Ctrl+S
  3. Nasadit → Nové nasazení → Webová aplikace
  4. Spustit jako: Já  ·  Přístup: Kdokoli (ne "Kdokoli přihlášený")
  5. Klikni Nasadit → zkopíruj URL → vlož do pole výše
⚠ Po každé úpravě kódu: Nasadit → Spravovat nasazení → tužka → Nová verze → Nasadit
